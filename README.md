# DSS - Weighted Product

This repo is an example of how to implement DSS using weighted product method using Laravel

## Requirement

- php <= 7
- mysql 5.3
- composer

## Installation
- clone this repo
- import the DB
- locate terminal into the place u clone this repo, then run
```
php artisan serve
```
